Evaluating Machine Learning Techniques for Website Fingerprinting Attacks

Independent Study with the Secure, Private Internet Research Group (SPIN)

  • Pursued an independent study to evaluate the assumptions made by different machine learning models for targeting Website Fingerprinting (WF) attacks on the Tor network.

  • Existing models include Naïve Bayes, Decision Trees and SVM, but they all employ certain restrictions (e.g., user visits only a small set of web pages, a single tab whose content is unchanging) that aren’t realistically attainable. Evaluated the assumptions made by such models and the performance when these assumptions are violated.

  • Proposed two neural network models (a single-layer network as well as a 1D CNN) as a tool to effectively classify multi-tab, open-world, time-variant crawl data, and evaluate their performance against the state-of-the-art SVMs. Results are promising and merit further research.
