Natural Language Processing Course Project

Generating coherent summaries of long documents such as books is a relatively untouched field, with most of the research work being focused on short documents. In this Project, we looked at summarization of books which have a back-index as a novel technique for generating good summaries. We provided a new dataset for the evaluation of such summarization systems. We focused on extractive summarization using LexRank, an unsupervised learning algorithm based on eigenvector sentence centrality and compared it with the current state-of-the-art methods. Evaluating Machine Learning Techniques for Website Fingerprinting Attacks

Independent Study with the Secure, Private Internet Research Group (SPIN)

Pursued an independent study to evaluate the assumptions made by different machine learning models for targeting Website Fingerprinting (WF) attacks on the Tor network. Existing models include Naïve Bayes, Decision Trees and SVM, but they all employ certain restrictions (e.g., user visits only a small set of web pages, a single tab whose content is unchanging) that aren’t realistically attainable. Evaluated the assumptions made by such models and the performance when these assumptions are violated. Proposed two neural network models (a single-layer network as well as a 1D CNN) as a tool to effectively classify multi-tab, open-world, time-variant crawl data, and evaluate their performance against the state-of-the-art SVMs. Results are promising and merit further research.

Technologies: Keras, Numpy, Scikit-learn, Python, C++, Bash.
